🍕 Pizza Hut Sales Analysis using MySQL
📌 Project Overview

This project focuses on analyzing Pizza Hut sales data using MySQL to uncover valuable business insights. The dataset contains 10,000+ sales records,
and the analysis was performed using various SQL techniques to answer real-world business questions related to sales performance, revenue generation, 
customer behavior, and product trends.

The primary objective of this project is to demonstrate practical SQL skills and showcase how data can be transformed into actionable business insights.

📊 Dataset Information
Dataset: Pizza Hut Sales Data
Records: 10,000+ rows
Database: MySQL
Domain: Food & Beverage / Sales Analytics

🎯 Project Objectives
Analyze sales performance and revenue trends.
Identify top-selling pizzas and categories.
Understand customer ordering patterns.
Calculate key business metrics using SQL.
Practice and apply advanced SQL concepts on a real-world dataset.
🛠️ SQL Concepts Used

Throughout this project, various SQL functions and techniques were used, including:

Joins (INNER JOIN, LEFT JOIN)
Subqueries
Aggregate Functions (SUM, COUNT, AVG)
GROUP BY
ORDER BY
LIMIT
DISTINCT
Date and Time Functions
Revenue Calculations
Ranking and Sorting
